## Descripción
Sistema  para generar comprobantes bancarios con formato fijo. Permite cambiar dinámicamente el nombre de bancos, números de cuenta, montos y fechas.

## 🚀 Cómo usar

### Acceso online
Abre tu navegador en: 
```
https://vegatovararmandoemanuel-maker.com 
```

### En tu teléfono
1. Abre el navegador del móvil
2. Ingresa la URL anterior
3. El sitio se adapta automáticamente a la pantalla

### Pasos para generar documentos
1. **Completa los campos** en el formulario izquierdo
2. **Observa la vista previa** actualizarse en tiempo real
3. **Descarga la imagen** con el botón verde "⬇️ Descargar Imagen"
4. **O imprime** directamente con el botón azul "🖨️ Imprimir"
